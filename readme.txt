== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.
<ul>
<li>1. Go to the menu 'Plugins' -> 'Install' and search for 'Wiloke Vimeo Gallery'</li>
<li>2. Click 'install'.</li>
<li>3. Go to Post/Page or any custom post type (support editor) and click on  'Wiloke Vimeo Gallery' shortcode</li>
</ul>
